Installation of base R for Apple Mac OS

This manual walks you through the installation of the R statistical software for personal computers and laptops.

Please read this installation manual carefully before installing and follow the instructions point by point!

It is assumed that you have (at least) Mac OS X version 10.6 (Snow Leopard) or above. If you have Mac OS X version 10.4 (Tiger) or older, consider upgrading your Mac OS X; see How to upgrade (www.apple.com).

Now, we are ready for the installation:

  • Follow this link to download the file R-4.0.3.pkg.
  • After clicking on the link, the package installer downloads the file R-4.0.3.pkg onto your computer.
  • Once the file has been downloaded, click on the link and walk through the installation process (accept the default options for now.) which consists of the following steps:
    1. Choose your language: and click on next
    2. The Software License is shown; click on next
    3. Select components: Do not change the default settings; click on next
    4. Startup options: Do not change the default settings; click on next
    5. Select additional tasks: Do not change the default settings; click on next
  • Now, R is being installed...
  • Once R has been installed, we want to test if it runs properly


Testing R

Let's check whether base R has been properly installed.

  1. In the Applications folder, is there an R icon that looks like this ? (If not contact your statistics lecturer)
  2. Double-click on the R icon. A window titled 'R Console' (see figure, below) should pop up. This means R is up and running!

  3. Ok, let's quit the R program. To this end, go to "File" in the menu bar and select "Exit".
  4. A dialog pops up that asks us whether we want to save the workspace. Click on "Don't save".
